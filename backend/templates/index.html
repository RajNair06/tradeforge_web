TradeForge
Technical Analysis API

TradeForge provides real-time stock data, technical indicators, and interactive charting through a clean REST API. Data sourced from Yahoo Finance, cached with Redis for fast responses.

Real-time Data Technical Indicators Interactive Charts Redis Caching

Quick Start

GET /latest-features/TCS.BO
GET /historical-data/TCS.BO?start_date=2026-01-01&end_date=2026-06-21&offset=0&limit=100

API Endpoints

GET /latest-features/{symbol}

Get the latest technical indicators and price data for a stock symbol.

Parameters
{symbol} required Stock symbol (e.g. TCS.BO, AAPL)
Example Request
GET /latest-features/TCS.BO
Example Response
{
  "symbol": "TCS.BO",
  "latest_features": {
    "basic_info": {
      "price": 3105.70,
      "timestamp": "2025-11-13 12:49:46"
    },
    "technical_indicators": {
      "sma_20": 3033.66,
      "sma_50": 3031.65,
      "sma_200": 3326.27
    }
  }
}
GET /historical-data/{symbol}

Retrieve historical stock data with Redis caching and pagination support.

Parameters
{symbol} required Stock symbol
start_date required Start date (YYYY-MM-DD)
end_date required End date (YYYY-MM-DD)
offset optional Pagination offset (default: 0)
limit optional Pagination limit (default: 50)
Example Request
GET /historical-data/TCS.BO?start_date=2026-01-01&end_date=2026-06-21&offset=0&limit=100
Features
  • Redis caching (60 seconds)
  • Pagination support
  • Automatic data freshness
GET /plotting/{symbol}/live

Interactive Chart.js visualization with technical indicators and real-time data. Returns an HTML page.

Parameters
{symbol} required Stock symbol
start_date required Start date (YYYY-MM-DD). End date defaults to today.
use_cache optional Use cached data (default: true)
Example Request
GET /plotting/TCS.BO/live?start_date=2026-03-01
Features
  • Interactive Chart.js visualization with SMA 20, 50, 100, 200
  • Price lags (1, 2, 3 days) and current indicators
  • Redis caching (60 seconds)
  • Automatic end date (today)
